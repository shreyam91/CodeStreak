# CodeStreak 🚀  
### Build Consistency. Master Coding Interviews.

[CodeStreak Repository](https://github.com/shreyam91/CodeStreak)

CodeStreak is a platform designed to make coding interview preparation structured, engaging, and consistent.  
Instead of solving random problems and forgetting patterns later, CodeStreak helps users develop a daily habit of practicing Data Structures & Algorithms through streaks, progress tracking, guided learning paths, and pattern-based practice.

---

# ✨ Features

- 🔥 Daily coding streak system
- 📚 Structured DSA roadmap
- 🧠 Pattern-based learning approach
- 📝 Flashcards for quick revision
- 📈 Progress tracking dashboard
- 🎯 Blind 75 / LeetCode 75 focused practice
- 📊 Habit-building gamification
- 👥 Accountability & social motivation
- 📱 Clean and beginner-friendly UI

---

# 🛠️ Tech Stack

- **Frontend:** React.js  
- **Backend:** Node.js  
- **Database:** MySQL  
- **UI Framework:** Bootstrap  

---

# 📸 Project Vision

Most people preparing for coding interviews face the same problems:

- Doing random LeetCode problems
- Forgetting patterns after a few days
- No clear roadmap
- Difficulty staying consistent

CodeStreak solves this by transforming interview prep into a sustainable daily habit.

The goal is simple:

> Learn one pattern at a time.  
> Stay consistent.  
> Build long-term retention.

# 🚀 Getting Started

## 1️⃣ Clone the Repository

```bash
git clone https://github.com/shreyam91/CodeStreak.git
cd CodeStreak
```

---

## 2️⃣ Install Dependencies

### Frontend

```bash
cd client
npm install
```

### Backend

```bash
cd server
npm install
```

---

## 3️⃣ Setup Environment Variables

Create a `.env` file inside the server directory.

Example:

```env
PORT=5000
DB_HOST=localhost
DB_USER=root
DB_PASSWORD=yourpassword
DB_NAME=codestreak
JWT_SECRET=your_secret_key
```

---

## 4️⃣ Run the Application

### Start Backend

```bash
npm run server
```

### Start Frontend

```bash
npm start
```
